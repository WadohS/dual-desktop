# GNOME Extensions Submission Text

## Short Description

Dual-monitor clock overlay with configurable position, fonts, formats and automatic text color.

## Long Description

Dual Desktop adds a clean clock overlay to each monitor on GNOME Shell.

Features include:

- per-monitor positioning
- optional shared settings across monitors
- configurable fonts
- configurable time and date formats
- automatic black/white text color based on wallpaper brightness
- manual text color override
- adjustable second-line offset and spacing

The extension is designed to work well with multi-monitor wallpaper setups and can also act as the clock component of the broader Dual Desktop project.
